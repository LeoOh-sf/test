## 구현 기능 설명
CSV파일에서 넷플릭스 주식 가격과 관련된 데이터 항목을 추출하여 해당 데이터 추출 결과를 시각적인 그래프로 표현하여 해당 결과를 생성형 AI에게 결과에 대한 해설을 넘겨 결과를 첨부한 파일입니다.

## 학습 내용
Pandas와 Numpy를 통해 CSV파일의 데이터를 정제 및 가공하는 기초적인 방법과 정제된 데이터를 Matplot을 통해 시각화하는 작업을 배웠습니다. 해당 모듈들의 기초적인 기능과 구동 원리 그리고 해당 모듈들을 활용하여 작성된 코드들이 어떤 목적과 기능을 가지는지에 대한 대략적인 인사이트를 학습하였습니다.

## 느낀 점
Pandas와 Matplot은 굉장히 똑똑한 프로그램이라는 점에서 놀랐습니다. 파이썬 정도면 충분히 인간 친화적인 자연어 구조이지라는 생각에 만족하였는데 padas와 Matplot은 파이썬보다 많은 부분에서 생략된 문법들을 알아서 이해하고 출력해준다는 느낌을 받았습니다.
추가적으로 해당 모듈들의 유용성이 제 삶에서 여태까지 불필요한 시간 소모를 상당히 많았다는 부분을 깨달았습니다. Matplot이 저보다 훨씬 유용하고 깔끔하게 많은 량의 데이터를 시각화하였고, pandas는 제가 노력하여 뽑아낸 엑셀 자료보다 빠르고 효율적으로 데이터를 가공하였습니다. 남는 시간을 최대한 활용하여 해당 모듈들의 사용법을 익혀두고자 합니다.
